Git

  • git init - создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий Git.
  • git clone - клонирование репозитория с GitHub на компьютер
  • git add - добавление файлов в индекс
  • git status - проверка статуса репозитория
  • git commit -m - добавление файлов в репозиторий
  • git push - внесение изменений на репозиторий
  • git pull - получение изменений из репозитория на компьютер
  • git branch branch_name- создание новой ветки
  • git branch -m - переименование названия ветки
  • git branch - просмотр веток
  • git checkout - переключение между ветками
